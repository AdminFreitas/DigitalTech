Minha senha já vazou?

Confere se a senha já apareceu em vazamentos de dados conhecidos, usando a base pública Have I Been Pwned — sem nunca enviar sua senha real pela internet.

consultando base de vazamentos…
🔒 A senha é transformada em hash SHA-1 aqui no navegador. Apenas os 5 primeiros caracteres do hash são enviados à API — a senha completa nunca sai do seu dispositivo (técnica chamada k-anonimato).

Se ela já vazou: troque essa senha imediatamente em todos os lugares onde a usa e evite reutilizá-la — vazamentos alimentam ataques automatizados que testam a mesma senha em vários serviços.
