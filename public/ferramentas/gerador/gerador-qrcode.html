Transforma qualquer texto, link ou dado em um QR Code gerado direto no seu navegador — nada é enviado para servidores externos.

Como funciona: a codificação acontece localmente, usando uma biblioteca JavaScript aberta — o conteúdo do QR Code não passa por nenhum servidor. Funciona bem para links, texto, Wi-Fi (formato WIFI:S:rede;T:WPA;P:senha;;) ou vCards.
